    环境搭建

    1、环境变量配置

    PUB_HOSTED_URL = https://pub.flutter-io.cn
FLUTTER_STORAGE_BASE_URL = https://storage.flutter-io.cn

    2、下载 Flutter

    git clone -b beta https://github.com/flutter/flutter.git

    3、用户变量
    Path 路径：控制面板 -> 系统和安全 -> 系统 -> 高级系统设置 -> 环境变量，Path 变量中添加 Flutter 路径，到 \bin 级

    常见问题

    • flutter doctor sdk 版本问题
      升级一下即可

    • 缺少 flutter dart 插件
      安装下插件即可

    • Exception in thread main java.lang.NoClassDefFoundError
      方案一：flutter 使用的 JDK 版本为 1.8 安装 JDK 1.8 即可
      方案二：缺失库

    • Some Android licenses not accepted.

    • Android license status unknown
      flutter doctor –android-licenses 一路Y

    • 真机测试，一直卡在 loading
      在安装flutter的路径底下进入bin/cache，然后把lockfile删除
