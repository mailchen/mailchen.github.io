    状态的存储访问统一使用 get()、set()、sync()、call() 方法

    demo

    安装

    npm i vuex-pathify

    配置

    scheme path state getter mutation action notes
    standard /foo foo foo SET_FOO setFoo 默认
    simple /foo foo foo foo setFoo 简洁

    参数说明

    deep:

    0：禁止访问子属性
    1：启用对现有子属性的访问
    2：允许创建新的子属性

    自定义配置

    formatters:

    camel - camelCase
    snake - snake_case
    const - CONST_CASE
