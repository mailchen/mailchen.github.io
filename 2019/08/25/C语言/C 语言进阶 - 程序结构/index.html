    全局变量

    全局的生命周期，作用域
    与任何函数无关，任何函数都可以使用全局变量

    未初始化的全局变量，默认值0，指针指向NULL
    只能编译时刻已知的值来初始化全局变量
    初始化发生在main函数之前
