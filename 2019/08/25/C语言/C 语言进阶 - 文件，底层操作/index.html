    格式化

    printf：%[flags][width][.prec][hlL]type
scanf： %[flags]type

    • flags

       -：左对齐
      +：强制+
      0：0填充；

    • width

      number
      .number
      *
      .*

    • HlL

      hh char
      h short
      l long
      ll long long
      L long double

    • type

      i\d -> int
      g\G -> float
      u -> unsigned int
      o -> 八进制
      a\A -> 十六进制浮点
      x\X -> 十六进制\字母大写的十六进制
      c -> char
      s -> 字符串
      P -> 指针
      f\F -> float
      e\E -> 指数
      n -> 读入\写出的个数

    返回值

    printf：读入的项目数
scanf：输出的字符数
