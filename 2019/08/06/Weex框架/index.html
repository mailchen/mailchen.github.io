    环境搭建

    1、Node.js 安装
    2、npm install -g weex-toolkit
    3、weex -v //查看 weex 版本
    4、weex //查看帮助
    5、weex create 项目名（项目名只能小写，安装过程类似 Webpack 安装 Vue 项目奖）
    6、添加 Android 或 iOS 平台：weex platform add android 或 weex platform add ios
    7、weex run android/ios
    8、weex debug 调试
    9、weex playground app 扫描二维码

    weex 组成

    Weex由多个关键模块组成，分别是DSL transformer、JS Framework、HTML5/iOS/Android Renderer和工具链

    transformer

    • HTML 解析工具：htmlparser
    • CSS 解析工具：cssom
    • JavaScript 解析工具：uglify-js
